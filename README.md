# Sysprite

A tiny animated pet for your macOS menu bar that reflects live system pressure. Open-source spiritual cousin of RunCat — but driven by CPU + memory + disk + network combined, with a built-in stats menu, an `Activity Monitor` shortcut, and a JSON CLI for [sketchybar](https://github.com/FelixKratz/SketchyBar) plugins.

- 🐾 Animated menu-bar sprite — speed maps to combined system pressure (CPU, memory, sustained network, near-full disk)
- 🔢 Optional `42%` pressure label in the menu bar, with color tint at 70% / 90%
- 📊 Click the icon to see CPU / Memory / Disk / Network (and Battery on laptops) with live 60-sample sparklines
- 🎨 6 bundled themes (cat, parrot, horse, dog, rabbit, snake) + drop-in folders for your own art
- 🔔 Optional high-CPU notifications, configurable threshold
- 🚀 Launch at login via `SMAppService`
- 🪟 SwiftUI preferences pane
- ⌨️ `Open Activity Monitor` menu item
- 🧩 `sysprite stats [--json | --cpu | --memory | --pressure | ...]` CLI for sketchybar / scripts
- 📦 Pure Swift Package — no Xcode project

## Install

```bash
git clone https://github.com/AbhinavGupta-de/sysprite
cd sysprite
make install     # builds, packages .app, copies to /Applications
open /Applications/Sysprite.app
```

Requirements: macOS 13+, Swift 5.9+ (Xcode Command Line Tools).

## Build / develop

```bash
swift build           # debug
swift test            # XCTest suite
make app              # produces .build/Sysprite.app
make run              # build + open
make sprites          # regenerate bundled theme PNGs
make icon             # regenerate Resources/AppIcon.icns
make release-zip      # .build/Sysprite.zip for GitHub releases
```

## How "pressure" works

```
pressure = max(cpu%, memory%)
        + 75% floor if disk usage ≥ 90%
        + 0.7 × network% if network ≥ 50% of reference
```

Animation interval ranges from **500 ms/frame** (idle) to **20 ms/frame** (saturated) on an exponential curve — the difference is visually obvious. Reference throughput defaults to 5 MB/s.

## Themes

Each theme is a folder of zero-padded PNG frames under `Resources/Themes/<name>/`. They're loaded as **template images** so macOS auto-tints them to match the menu bar. Drop a folder and rebuild — that's it. The bundled six are 100% original art generated by [`scripts/generate_sprites.swift`](scripts/generate_sprites.swift) (MIT-licensed alongside the code).

See [`Resources/Themes/README.md`](Resources/Themes/README.md) for the frame format.

## Sketchybar integration

The running app writes a JSON snapshot to `~/Library/Application Support/Sysprite/snapshot.json` every second. Plugins read it via the CLI:

```bash
sysprite stats --json       # { "cpu": …, "memory": …, "pressure": …, "net_down_bps": …, … }
sysprite stats --cpu        # "42"
sysprite stats --pressure   # "82"
sysprite stats --battery    # "31"
```

A drop-in plugin lives in [`integrations/sketchybar/`](integrations/sketchybar/) — see its README.

```bash
sudo ln -sf /Applications/Sysprite.app/Contents/MacOS/Sysprite /usr/local/bin/sysprite
cp integrations/sketchybar/sysprite.sh ~/.config/sketchybar/plugins/
# then add the snippet from sketchybarrc.example
```

## CLI reference

```
sysprite stats [flag]
  --json         Full snapshot
  --text         One-line summary (default)
  --pressure     Combined pressure %
  --cpu          CPU %
  --memory       Memory %
  --disk         Disk %
  --net-down     Download bytes/sec
  --net-up       Upload bytes/sec
  --battery      Battery %
  --help, -h     Show help
```

## What's not in here (yet)

- GPU usage — no public API on Apple Silicon for system-wide GPU%, would need IOReport (private).
- Top processes list à la `htop`.
- Per-disk-volume rows.
- Notarized signed distribution (you'll see a Gatekeeper prompt on first launch since releases are ad-hoc-signed).
